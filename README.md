 Audioteca Gamer

A Audioteca Gamer é um projeto web interativo que simula um computador retrô dos anos 80, permitindo que o usuário digite o nome de um jogo e escute suas trilhas sonoras.

 Funcionalidades

- Interface com estética retrô inspirada em sistemas DOS.
- Busca por jogos digitando o nome na tela inicial.
- Exibição da capa do jogo na tela da TV.
- Reprodução de trilhas sonoras através de botões interativos.
- Animação visual da TV ao tocar as músicas.
- Transição suave entre as telas.

 Jogos Disponíveis

- **Streets of Rage**
- **Sonic the Hedgehog**
- **Resident Evil**

 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts (VT323)
